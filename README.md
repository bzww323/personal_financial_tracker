# personal_financial_tracker
Личный финансовый трекер для анализа своих расходов и доходов. 
Также присутствует ml-фича  - линейная регрессия, которая предсказывает расход пользователя на следующую неделю
Если по каким-либо причинам вы будете использовать этот проект в своих целях, то создайте на своем пк в корневой папке текстовый файл формата ENV и названием .env, и напишите в нем SECRET_KEY=ваш_секретный
Секретный ключ надо будет придумать самому
